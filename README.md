# 🎮 Tic Tac Toe Game

A simple and interactive **Tic Tac Toe** game where two players can compete against each other.

## 📌 Features
- 🔹 Two-player mode  
- 🔹 User-friendly interface  
- 🔹 Detects win, loss condition  
- 🔹 Option to restart the game
